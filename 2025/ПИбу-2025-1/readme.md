задание:  

Написать функцию перевода из римской в десятичную систему счисления:  

Значения римских цифр:  

```txt
'I': 1,
'V': 5,
'X': 10,
'L': 50,
'C': 100,
'D': 500,
'M': 1000
```

Тесты:  
```txt
IV → 4
VIII → 8
CM → 900
XC → 90
MMMM → 4000
MCMXCIV → 1994
LVIII → 58
```
